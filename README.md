# Digital Clock
## LINK: https://sumananand460.github.io/Digital_Clock/

A simple digital clock built with HTML, CSS, and JavaScript.

---

## Files
```
digital-clock/
├── index.html    → Clock structure
├── style.css     → Styling and animations
└── script.js     → Clock logic
```

---

## How to Run

1. Download or clone the project
2. Open `index.html` in your browser

That's it — no installation needed.

---

## Features

- Shows live time in 12-hour format
- Displays AM / PM
- Shows the current date
- Blinking colon between hours, minutes, and seconds

---

## Built With

- HTML
- CSS
- JavaScript
